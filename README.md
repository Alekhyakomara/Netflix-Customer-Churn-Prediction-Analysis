Netflix Customer Churn Prediction using Machine Learning
📌 Project Overview

Customer churn is one of the biggest challenges faced by subscription-based businesses like Netflix. This project develops a Machine Learning model to predict whether a customer is likely to cancel their subscription based on customer demographics, subscription details, and viewing behavior.

The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, model evaluation, and performance comparison using multiple classification algorithms.

🎯 Objectives
* Analyze customer behavior and subscription patterns.
* Perform data cleaning and preprocessing.
* Explore customer churn trends using EDA.
* Build multiple Machine Learning classification models.
* Compare model performance using evaluation metrics.
* Predict customers who are likely to churn.
* Generate business insights to improve customer retention.
